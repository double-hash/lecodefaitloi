## Le code fait loi

3D (printable) book virtual objectboilerplate (paged.js, threeJS, gulp)

### Installation

`$ npm install`

### Compiling JS, CSS (in ./public/assets/) and autoreload:

`$ gulp front (task file located in ./gulp-tasks)`

Gulp task file : ./gulp-tasks/front.js

### Dev

#### JS/CSS folder

#### HTML pages
